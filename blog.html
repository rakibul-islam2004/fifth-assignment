What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing scripts to manipulate the content, structure, and style of a webpage.

How do you select an element from the Dom?

To select an element from the DOM, you can use methods like document.getElementById() for specific IDs, document.querySelector() for CSS selectors, or document.getElementsByClassName() and document.getElementsByTagName() for classes and tags, respectively. These methods allow you to easily access and manipulate elements within the HTML structure.

What is event delegation in the context of DOM, and why is useful?

Event delegation is a technique in DOM manipulation where a single event listener is added to a parent element instead of multiple listeners to child elements. This is useful because it reduces memory usage and improves performance, especially with dynamic content. When an event occurs, the parent can catch it and determine which child triggered it, simplifying event management and enhancing maintainability.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes and styles in the DOM using element.setAttribute('attributeName', 'value') to set attributes and element.getAttribute('attributeName') to retrieve them. For styles, use element.style.propertyName = 'value' to change individual CSS properties or element.style.cssText = 'styles' for multiple styles at once. This allows for dynamic changes to an element's appearance and behavior.
